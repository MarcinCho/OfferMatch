# Service Match

An app utilizing microservices architecture.


## Tech Stack

**Companies_MS:** Java, Spring Boot 3,

**Project_MS:** Java, Spring Boot 3,

**Client_FE:** TypeScript, React, Tailwind CSS, PrimeFaces (UI) 

**Databases and other**
- Mongo DB (Project_MS)
- PostgreSQL (Companies_MS)
